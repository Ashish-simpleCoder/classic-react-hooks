## Features

-  Comes with treeshaking
-  Typescript support
-  Small bundle size
-  Minimal and Easy to use

## Installation

For npm users

```bash
$ npm install classic-react-hooks
```

For pnpm users

```bash
$ pnpm install classic-react-hooks
```

For yarn users

```bash
$ yarn add classic-react-hooks
```
